Rebranding Project

This design project was undertaken with the goal of creating a style guide for Juventus FC's new brand identity.

Tools:

  1. Adobe Illustrator: This software was used to create the logos, illustrations, and other vector graphics in the document. Specifically to create primary and secondary logos and to create the visual elements in the document.

  2. Adobe InDesign: This software was used to create the typography hierarchy and to ensure the document is visually pleasing and easy to read.

  3. Adobe Photoshop: This software was used to edit and enhance photographs and other raster images in the document and to create the color palette.

Methadologies:

  1. Typography: Worked with primary and secondary typefaces, and guidelines for typography hierarchy. Discovered how to select appropriate typefaces and how to use typography to create a clear hierarchy of information.

  2. Colour Palette: Used the hex codes, RGB, and CMYK of the color palette. selected a cohesive color palette and learned the different ways to specify colors.

  3. Logos: Created primary and secondary logos, in color variations, monotone black and white, and the guidelines for styling. Designed effective logos while considering factors such as spacing, color, size, and readability.
